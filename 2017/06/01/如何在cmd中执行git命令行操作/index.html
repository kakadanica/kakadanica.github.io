如何在cmd中执行git命令行操作

此情况发生场景


安装git时未勾选Windows Explorer integration,在需要操作git命令行时，找不到操作面板

解决方案：


配置环境变量

1) 新建系统变量 GIT_HOME:[git安装地址（例如：D:\Program Files\Git)]

2) 编辑Path,新建“%GIT_HOME%\usr\bin”和“%GIT_HOME%\bin”

结果


打开cmd,任意使用git命令进行需要的操作
